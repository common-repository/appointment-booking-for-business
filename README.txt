== Description ==

Appointment Booking For Business is a **desktop and mobile-friendly** booking calendar plugin featuring a simple solution for **accepting online appointments** from your customers. Your visitors can easily select and book available time slots determined by you. Our plugin now includes free Stripe payment gateway.

Appointment Booking For Business includes an easy-to-use interface that allows you to create and insert a beautiful, clean calendar on your website within minutes. Accept payment through Stripe for your bookings.

[Website](https://www.bookingforbusiness.app) | [Live Demo](https://demo.bookingforbusiness.app) | [Documentation](https://www.bookingforbusiness.app/support)

### Features: 

* Easy setup for beginners and non-technical users
* Fits a wide variety/range of business types
* Create a calendar and start receiving bookings within minutes
* Stripe payment gateway
* Simple to use interface
* Mobile-friendly with an intuitive vertical calendar on mobile devices
* Create an unlimited amount of calendars and extras
* Customers can book available time slots determined by you
* Set time slot duration
* Block unavailable dates in the calendar up to one year in advance
* Set available and unavailable times
* Ability to set a price for booking
* Ability to link extra charges and services for your customers to add-on to their bookings
* Email notification to the customer and admin when a booking is made
* Email notification to the customer when a booking is canceled
* Easily manage and sort confirmed bookings from the dashboard
* Insert calendar into pages or posts with simple shortcode
* Multi-Site support
* Tested with WordPress 5.0 and Gutenberg editor
 
== Installation ==
 
To install **Appointment Booking For Business**, follow these steps:

1.	Download and unzip the Appointment Booking For Business plugin.
2.	Upload the entire booking-for-business/ directory to the /wp-content/plugins/ directory.
3.	Activate the Appointment Booking For Business plugin through the Plugins menu in WordPress.
4.	Configure and create a calendar from the left dashboard menu Booking For Business >> Calendars
5.	To insert an appointment calendar just copy the shortcode from the Calendars subpage and paste the shortcode into any page or post where you would like the calendar to appear.
 
== Frequently Asked Questions ==
 
= Where can I find documentation for the plugin? =
 
You can find detailed documentation on our website at [https://www.bookingforbusiness.app/support](https://www.bookingforbusiness.app/support)
 
= Is there a demo for the plugin? =
 
Yes, there is a live demo available at [https://demo.bookingforbusiness.app](https://demo.bookingforbusiness.app). You can demo the dashboard settings and booking form.

= How can I set my timezone for bookings? =
 
You can set and change the timezone by going to Settings >> General Settings in the WordPress dashboard.

= Where can I set my email address for new booking notifications? =

By default, the plugin will send all new booking notifications to the admin email address set within Settings >> General Settings in the WordPress dashboard.

= How can I accept payments for my bookings? =

Yes, we offer free Stripe payment gateway integration. We will be adding Paypal integration shortly in an upcoming update.

= Can I request a new feature? =

Yes! We would love to hear new feature suggestions from our users for future versions. Please email us at [support@bookingforbusiness.app](mailto:support@bookingforbusiness.app) to request a feature.
